Gnome-VFS PLP Module

This is a module for the Gnome-VFS abstracted filesystem that uses libplp to access files on Psion palmtops. With this installed you can use Nautilus and other programs that use the Gnome-VFS library to access your palmtop.

Requires Gnome-VFS and plptools.
