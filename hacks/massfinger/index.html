massfinger

I wrote this to be a more powerful version of a script called pfinger which I found on cslib.

This is a tool to finger a large number of machines and display a list of who is logged into them. I designed it as a kind of status monitor for the Southampton Uni ECS Computer Labs. This is not a security scanner although I imagine it does generate quite a bit of network traffic if several people are running it at once. Don't blame me if your network admins get annoyed.

Features:

  • Multiprocess fork()ing for faster scanning
  • Scanning Unix/Linux machines running fingerd
  • Scanning Windows machines using code from samba
  • Gtk+ and console mode displays.
