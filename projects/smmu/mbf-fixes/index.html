MBF fixes

This is a fixed-up version of the source code to Marine's Best Friend, that includes:
  • License change to the GPL (the last version of MBF was released before the Doom source was relicensed under the GPL.
  • Some bug fixes
  • Compilation fixed under later versions of Allegro.
The full change history is here. You can also browse it on GitHub.

Note that I made these changes almost 8 years ago from the time of writing (2009), so this may still not compile with current versions of DJGPP/Allegro. I haven't tried.

Compare and contrast with WinMBF.
