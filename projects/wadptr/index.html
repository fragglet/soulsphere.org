wadptr - WAD compressor

wadptr is a utility for reducing the size of Doom WAD files. The "compressed" WADs will still work the same as the originals. The program works by exploiting the WAD file format to combine repeated/redundant material.

The source code is portable and will run on most Unix systems as well as Microsoft Windows. The program is Free Software under the GNU GPL v2+.

There is a manual page that describes how to use the program.

For the latest developments, you can follow the project on Github.
